Week 8 - CSS Responsive layout

Posted on November 24th, 2023

In week 8, I continued on improving the website by making the layout responsive.

I learnt and tested 2 different layouts compared to the grid layout we previously covered.

The flexbox layout with a card interface

This layout displays flexible cards in columns that can change depending on the screens resolution

This allows the user to view the website on any device and it will still look good

The multi-column layout

This layout displays the content in columns that can change depending on the screens resolution

This allows the user to view the website on any device and it will still look good

Both of these layouts are responsive and can be used on any device

Although neither are used on the learning journal page, they could be useful for the tutorial page
